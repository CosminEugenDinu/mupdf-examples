# mupdf-examples
Examples of using mupdf (fitz) library.

## Install mupdf

```sh
git clone https://github.com/CosminEugenDinu/mupdf-examples.git
cd mupdf-examples
chmod +x install-mupdf.sh
sudo ./install-mupdf.sh
```
